# S3 Bucket Lifecycle Policy using Terraform
## Run Locally

Clone the project

```bash
  git clone https://github.com/zablon-oigo/aws-s3-bucket-policy-terraform.git
```

Go to the project directory

```bash
  cd  aws-s3-bucket-policy-terraform
```

Initialize terraform

```bash
  terraform init
```
